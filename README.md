# Bootcamp Study Guide

## Description

This study guide helps the new student to go through the Prework for Coding. It introduces HTMl, CSS, Git and JavaScript, help understand the basics for coding, and work with them to build a project.

## Usage

Read through the topic to know about HTML, CSS, Git and JavaScript, you can find notes about each topic. Open the Chrome DevTools by pressing Command+Option+I (macOS) or Control+Shift+I (Windows) or right click on the page and click "Inpect" and then "Console. You can see what is covered from the prework modules.
